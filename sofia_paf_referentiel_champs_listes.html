Sofia-PAF — Référentiel champs, sélecteurs et listes

Fichier généré à partir des HTML Sofia-PAF fournis : page DISPOSITIF et page MODULE.

Usage : fournir ce fichier à Codex ou t’en servir pour coder le script Tampermonkey. Les listes déroulantes ci-dessous reprennent les options réelles extraites des balises <select> Sofia.

Résumé technique

Formulaire dispositif
#gdisp-form
Action source : /sofia/gdisp/update/559
Formulaire module
#gmodu-form
Action source : /sofia/gmodu/update/2541
Créer dispositif
/sofia/gdisp/create?pid=${Gdisp_co_them}
Le pid est le code thème.
Créer module
/sofia/gmodu/create?pid=${ID_DISPOSITIF}
Le pid est l’ID interne du dispositif créé.
