🧭 Outil unifié – Intention de parcours & Visualisation globale

Tutoriel – Bien démarrer avec l’OVP

Objectif : aller de l’intention → au dashboard → au déploiement → à la communication.

Check-list rapide (2 minutes)
  • Ouvre un projet existant (📂) ou commence un nouveau plan.
  • Ajoute 1 ou plusieurs intentions dans Saisie d’intention.
  • Vérifie les coûts et répartitions dans Dashboard global.
  • Importe l’Excel dans Déploiement, mappe les colonnes, auto-associe.
  • Analyse en masse dans Déploiement global.
  • Génère un HTML Drupal dans Communication.
  • Exporte le plan en Excel (📤) si besoin.

1) Saisie d’intention

Renseigne le titre, l’objectif, le contenu, l’accessibilité, les prérequis…

Complète les chiffres : effectif, présentiel, distanciel, durée, niveau, priorité, frais exceptionnels.

👉 Clique ➕ Ajouter au plan. Tu peux ensuite éditer/supprimer dans “Intentions ajoutées”.

2) Dashboard global

Contrôle :

  • Nombre total de parcours
  • Coût total + coût moyen / jour-stagiaire
  • Répartition des coûts, types de déploiement, P vs D

Le coût estimé inclut aussi les frais exceptionnels si renseignés.

3) Déploiement (Excel ↔ Plan)

Importe le fichier .xlsx, puis :

  1. Mappe les colonnes (dispositif, groupe, session, statut, durée, effectif, modalité).
  2. Ajuste les règles modalité → présentiel/distanciel (et le ratio mixte).
  3. Auto-associe les dispositifs Excel avec les intentions du plan.

4) Déploiement global

Vue multi-dispositifs : filtres + cases à cocher pour comparer en masse.

Tu peux repérer rapidement :

  • les dispositifs en retard
  • les écarts d’heures (Excel vs JSON)
  • les écarts d’effectif

5) Communication (Drupal)

Générer produit un HTML Drupal à partir des intentions chargées.

Tu peux :

  • générer tout le plan ou une intention
  • éditer dans l’éditeur visuel
  • copier ou télécharger le HTML
  • coller/importer un HTML OVP existant et le convertir
Sauvegarde / reprise

Utilise 💾 Sauvegarde projet pour conserver ton travail et 📂 Ouvrir / ajouter projet pour le restaurer.

Tu peux aussi importer des JSON d’intentions si tu as des exports existants.

Astuce : clique sur “Aller” pour te téléporter sur l’onglet correspondant.

📝 Intention de parcours

Informations

Exemple :

Contenu : Évaluation de la séquence pédagogique.

Mise en place : les stagiaires concevront une évaluation finale (sous forme de contrôle, projet ou exposé) qui permettra de vérifier qu'ils ont bien assimilé les concepts clés abordés tout au long de la séquence, en mobilisant les connaissances et compétences développées.

Ajoute un second bloc (niveau + modalité + durée + effectif) intégré au calcul budgétaire.

Exemple : 3h présentiel académique (48 pers) + 6h présentiel départemental (48 pers)

Astuce : tu peux sélectionner plusieurs fichiers HTML à la fois. Ils seront ajoutés au plan.
